Quartz-torrent

Ruby bittorrent library.

Download this project as a .zip file Download this project as a tar.gz file

Getting Started

First, download the ruby gem

quartz_torrent-0.0.1.gem

Install it

gem install quartz_torrent-0.0.1.gem

Requirements

This library has been tested with ruby1.9.1.

Running the curses client requires rbcurse-core (0.0.14).
